# ReactLab — Named Reactions Explorer

ReactLab is an interactive educational platform designed to help students and chemistry enthusiasts understand the mechanistic principles behind classic named reactions in organic chemistry.

![ReactLab Logo](https://img.shields.io/badge/ReactLab-Chemistry-blue?style=for-the-badge&logo=react)

## ⚗️ Project Overview

Whether you're preparing for JEE, NEET, university finals, or just passionate about chemistry, ReactLab makes complex reactions intuitive and visual. It moves beyond textbook diagrams by providing real-time 3D models and interactive simulations.

### 🔬 Key Features

- **114+ Named Reactions**: Comprehensive database covering Organic, Addition, Substitution, Elimination, Condensation, Rearrangement, and Industrial reactions.
- **Step-by-Step Mechanisms**: Detailed breakdowns of each reaction pathway, showing intermediate steps and electron flow concepts.
- **Interactive 3D Molecular Viewer**: Powered by **Three.js**, allowing users to rotate, zoom, and watch bond formation animations live in the browser.
- **Reaction Condition Simulator**: Adjust temperature, pressure, and concentration sliders to see how reaction outcomes and kinetics change in real-time.
- **Side-by-Side Comparison**: A dedicated mode to analyze two different reactions simultaneously, highlighting differences in catalysts, conditions, and mechanisms.
- **Mobile-Friendly Design**: Fully responsive UI built with modern CSS (glassmorphism, CSS variables, and smooth animations).

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla CSS with Glassmorphism), Javascript (ES6)
- **3D Rendering**: [Three.js](https://threejs.org/)
- **Typography**: Google Fonts (Inter, Orbitron, Fira Mono)
- **Icons**: Emoji-based and custom CSS icons

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your browser. No build process or server-side setup is required.

```bash
# Clone the repository
git clone https://github.com/kaustubhmohan-beep/ReactLab.git

# Navigate into the directory
cd ReactLab

# Open index.html (on Windows)
start index.html
```

## 📂 Project Structure

- `index.html`: The main entry point and single-page application structure.
- `css/style.css`: Modern, responsive design system with a premium dark-mode aesthetic.
- `js/data.js`: The central database of named reactions and their metadata.
- `js/app.js`: Main application logic, state management, and DOM interaction.
- `js/viewer3d.js`: Custom Three.js wrapper for molecular rendering and animation.
